Ada Salam Nih

Gunakan salinan Font Awesome Pro ini dengan bijak. Sangat ditujukan bagi kalian-kalian yang ingin menggunakannya untuk kepentingan/kebutuhan pribadi saja dan/atau apabila kalian memang dalam kondisi yang tidak memungkinkan untuk mendapatkan lisensi sendiri.

Jika ingin digunakan untuk tujuan publikasi dan komersil seperti web sekolah, perusahaan, biografi, toko online, atau web yang bersifat publik dan dapat diakses oleh bukan hanya pembuatnya maka silakan bersedia untuk membeli lisensi untuk kalian sendiri pada web resmi Font Awesome.
