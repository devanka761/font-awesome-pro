Penggunaan Link Tag


Salin salah satu kode Link Tag yang telah disediakan di atas dan letakan kode tersebut di antara tag <head></head>.

<head>

   Letakan salah satu kode link tag di sini 

</head>

Dan sudah bisa digunakan hehe

Penggunaan Import CSS


Salin kode Import CSS yang telah disediakan di atas dan letakan kode tersebut pada bagian atas di dalam file css atau bisa juga di antara tag <style></style>.

<style>
   Letakan kode import css di sini 

</style>

Dan sudah bisa digunakan hehe

Mungkin Berguna

Font Awesome Pro ini merupakan salinan resmi dari akun yang saya miliki. Penerapannya mirip seperti Font Awesome biasa (free), tetap bisa menggunakan class-class yg sama. Tetapi, ada tambahan-tambahan fitur yang tidak terdapat pada Font Awesome Free. Seperti Duotone, Light, Thin, Symbol Premium, Beta Premium, dan lain-lain
